# Rahul Bandikolla Portfolio

## Overview
Personal portfolio website for Rahul Bandikolla, an AI/ML Engineering Student specializing in Applied Machine Learning & Data Science.

## Tech Stack
- React 18 with TypeScript
- Vite for build tooling
- TailwindCSS for styling
- Shadcn UI components
- Framer Motion for animations
- React Router for navigation

## Project Structure
- `/src` - Source code
  - `/components` - React components
  - `/pages` - Page components
  - `/hooks` - Custom React hooks
  - `/lib` - Utility functions
- `/public` - Static assets

## Running the Project
The application runs on port 5000 using Vite dev server:
```
npm run dev
```

## Deployment
Configured for static deployment. Build output goes to `/dist`.
